## Русский

### О проекте

Персональный сайт-визитка будущего веб-разработчика. Создан в учебных целях для практики frontend-разработки.

### Функционал

- **Стеклянный дизайн (Glassmorphism)** — полупрозрачные карточки с размытием фона
- **Биография при наведении** — всплывающее окно с информацией при наведении на фото
- **Прогресс-бары навыков** — визуальное отображение уровня владения HTML, CSS, JavaScript
- **Счётчик лайков** — интерактивный счётчик с сохранением в localStorage
- **Кнопки соцсетей** — ссылки на GitHub, ВКонтакте, Telegram
- **Смена темы** — переключение между светлой и тёмной темой
- **Анимации** — плавные эффекты при скролле и наведении

### Технологии

| CSS | 70% |

### Как запустить

1. Склонируйте репозиторий:
```bash
git clone https://github.com/Perfomanto/my-first-site.git
```

2. Откройте `index.html` в браузере

### Структура файлов

```
my-first-site/

├── style.css     # Стили (CSS)

└── me.jpg        # Фото профиля
```

### About

Personal portfolio/business card website of a future web developer. Created for learning frontend development purposes.

### Features

- **Glassmorphism design** — translucent cards with background blur
- **Bio on hover** — popup with information when hovering over the profile photo
- **Skills progress bars** — visual representation of HTML, CSS, JavaScript proficiency
- **Like counter** — interactive counter with localStorage persistence
- **Social buttons** — links to GitHub, VK, Telegram
- **Theme toggle** — switching between light and dark themes
